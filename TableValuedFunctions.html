Table Valued Functions

As the name implies a Table Valued Function returns a table of results.  An example of a table valued function is created is:

 

CREATE FUNCTION dbo.fn_GetAllEmployeeOfADepartment(@DeptID AS INT) 

RETURNS TABLE

AS

RETURN

(

SELECT * FROM Employee E

WHERE E.DepartmentID = @DeptID

)

GO
