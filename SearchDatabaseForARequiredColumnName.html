Search Database for a Required Column Name

This has proved to be one of the all-time most useful script when analysing the database.

 

SELECT COLUMN_NAME AS 'ColumnName'

,TABLE_NAME AS  'TableName'

FROM INFORMATION_SCHEMA.COLUMNS

WHERE COLUMN_NAME LIKE '%[MyName]%'

ORDER BY TableName

,ColumnName;
